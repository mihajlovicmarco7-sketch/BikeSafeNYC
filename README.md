# BikeSafeNYC 🚲

## Description
BikeSafeNYC is a web application designed to educate cyclists on bike laws, safety practices, and advocacy opportunities.

## Features
- View bike laws in plain English
- Compare laws across states
- Learn safety tips
- Promote cyclist awareness

## Technologies Used
- Node.js
- Express
- Handlebars
- CSS

## How to Run
1. npm install
2. npm start
3. Visit http://localhost:3000
