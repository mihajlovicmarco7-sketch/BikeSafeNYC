# Local Development

## MongoDB Setup
To locally test/develop with MongoDB, use the following setup:

* Collection Name: `BikeSafeNYC`

### Seed data

Run `npm run seed` in your terminal. You should expect to see an output of: `Seeded data!!!` if all was succesful.
